# Dataset Setup

The dataset for this project is hosted on Google Drive to keep the repository lightweight and modular.

## Requirements

This project uses Pixi for environment and dependency management.

## Usage

To download and extract the dataset into the current directory, run:

```bash
pixi run python download_datasets.py https://drive.google.com/file/d/1-OQb2hs7wXwjGMQ11vDUsQ7NMPIT4vlV/view?usp=sharing
